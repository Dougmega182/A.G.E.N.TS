## 1. Purpose
Define physical security controls aligned to PSPF physical security requirements for facilities, assets, and records.

Scope
All physical locations, devices, and storage media used to run or store A.G.E.N.T.S. systems and records.

Principles
- Least access: only authorised personnel can access secure areas.
- Defence in depth: multiple layers of physical protection.
- Accountability: access is logged and audited.

Controls
1. Facilities and Zones
- Define security zones and access tiers for all locations.
- Maintain a register of approved facilities and zone classifications.

2. Access Control
- Access is granted on a need-to-know basis.
- Visitors require escort in secure zones.
- Access logs retained per Records Retention Schedule.

3. Asset Protection
- Hardware and storage media are inventoried and tracked.
- Secure storage required for removable media.

4. Environmental Security
- Protect systems from fire, flood, and power interruptions.
- Maintain tested backup and recovery procedures.

5. Incident Response
- Physical security incidents are reported under Protocols (RED/BLUE as applicable).
- Evidence preserved and reported to Audit Agent.

6. Disposal
- Physical disposal of media follows approved sanitisation methods.
- Disposal actions logged and retained.

Records
- Facility access logs
- Visitor logs
- Asset inventory
- Incident reports
- Disposal records
