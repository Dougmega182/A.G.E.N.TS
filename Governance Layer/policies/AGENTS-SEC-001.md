## 1. Purpose and Scope

To protect the Corporation's information, infrastructure, and agents from unauthorised access, disruption, or compromise. This policy aligns with the Essential Eight and ISM controls.

---

## 2. Application

This document applies to all agents, officers, and systems within the A.G.E.N.T.S. Corporation.

---

## 3. Legislative and Policy Framework

This instrument gives effect to the ASD Essential Eight maturity model and the Australian Government Information Security Manual (ISM).

---

## 4. Operative Provisions

### Physical Security
- Restricted access to the central memory core and compute clusters.
- All physical access must be logged and monitored.

### Network Security
- Internal network is air-gapped from untrusted external entities except via authorised gateways.
- Multi-factor authentication required for all elevated access.

### Agent Security
- Periodic integrity checks for all command-level agents.
- Zero-trust architecture for inter-domain communication.

### Incident Management
- Report security events to the Security Lead and Audit Agent immediately.
- Maintain incident response plans for rapid recovery.

---

## 7. Records Management and Retention

Administrative records created under this instrument (including security logs) are stored in `Governance Layer/`. Retention and disposal are governed by the ALRMF baseline.
