## 3. Overall Performance Rating Formula

Overall Performance Score (OPS) formula:
OPS = sum(Score_i * Weight_i)

| OPS Range | Rating | Description | Recommended Action |
|---|---|---|---|
| 95% - 100% | Outstanding | Consistently exceeds expectations | Resource uplift, promotion consideration |
| 85% - 94.9% | Exceeds Expectations | Strong performance with notable strengths | Recognition + modest resource increase |
| 75% - 84.9% | Meets Expectations | Solid, reliable performance | Standard resource allocation |
| 65% - 74.9% | Needs Improvement | Below target in key areas | Remediation plan + support |
| Below 65% | Unsatisfactory | Significant performance or integrity concerns | Formal review + possible reassignment |

Minimum Integrity Threshold: Integrity and Alignment must be >= 92% to be eligible for any rating above Meets Expectations.
